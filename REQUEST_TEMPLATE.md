# REQUEST TEMPLATE — スマホからの依頼テンプレート

スマホのメモアプリ・チャット・メールなどから、そのままコピーして使える依頼フォーマットです。  
**【】内を自分の内容に置き換えて** 送信してください。

---

## 基本テンプレート（コピー用）

```
【Smile AI Studio 開発依頼】

■ 日付
【例: 2026/07/09】

■ プロジェクト（1つ選択）
□ FuwafuwaPanicManager
□ GiftCanvas
□ イベント相談LP
□ その他【　　　　　】

■ 種別
□ バグ修正
□ 機能追加
□ 文言・デザイン変更
□ 確認・調査のみ
□ その他【　　　　　】

■ 優先度
□ 高（今日〜明日）
□ 中（今週中）
□ 低（急がない）

■ 依頼内容（具体的に）
【例: イベント相談LPの問い合わせボタンの色を青からオレンジに変えたい】

■ 期待する結果
【例: スマホで見たとき、ボタンが目立ってタップしやすい】

■ 変更してほしくないこと（あれば）
【例: フォームの項目や送信先は変えないでほしい】

■ 参考（URL・スクショ・文言案）
【あれば記載】

■ 連絡先
【名前 / 返信先】
```

## 依頼時のコツ

1. **プロジェクト名を必ず書く** — どのリポジトリか特定できます
2. **「変えないでほしいこと」も書く** — 既存機能を守りやすくなります
3. **スクショや URL があると早い** — 特にデザイン・レイアウトの依頼
4. **優先度を正直に** — 「高」は本当に困っているときだけ
5. **曖昧なときは「確認・調査のみ」** — 実装前に方針を一緒に決められます

---

## 依頼後の流れ

1. Smile AI Studio（または AI エージェント）が内容を確認
2. 必要なら追加質問
3. 開発 → テスト → GitHub Push → Netlify 確認
4. **開発完了通知** が届く
