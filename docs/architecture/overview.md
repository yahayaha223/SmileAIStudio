# アーキテクチャ概要

Smile AI Studio は **メタリポジトリ** として機能します。  
各プロダクトのソースコードは個別リポジトリに置き、このリポジトリでは以下を統括します。

- ビジョン・開発ルール
- プロジェクトメタ情報
- AI エージェント定義
- 依頼テンプレート・手順書
- 自動化スクリプト（将来）

## 設計方針

1. **単一の入口** — 依頼者も開発者も README から辿れる
2. **エージェント拡張** — `agents/` に設定を追加するだけで役割を増やせる
3. **プロジェクト拡張** — `projects/<name>/` を追加し PROJECTS.md を更新
4. **ルールの一元化** — DEVELOPMENT_RULES.md が全エージェントの最優先参照先
