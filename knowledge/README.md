# 会社の知識（knowledge）

株式会社えがおのきろく専用の知識ベースです。  
LINE AI秘書（チャッピー）は、OpenAIへ送る前にこのフォルダの Markdown を読み込みます。

## ファイル

| ファイル | 内容 |
|----------|------|
| `company.md` | 会社の基本情報 |
| `vision.md` | Mission / Vision |
| `projects.md` | プロジェクト |
| `events.md` | イベント |
| `people.md` | 人・役割 |
| `products.md` | 商品・サービス |
| `faq.md` | よくある質問 |
| `meeting.md` | 会議・判断メモ |
| `todo.md` | TODO |

## 回答ルール（AI）

- knowledge を一般知識より優先
- 矛盾時は knowledge を採用
- 無い事実は推測せず「まだ登録されていません」
